This is a sample project layout for the first few labs.

Please put ALL of your source files in the 'src' directory.


If there is error "Unknown target connected" when entering debug mode, do following:

- Select "Flash" / "Configure Flash Tools..."
- Click "Debug" tab
- Click "Settings" in the right hand column (next to "Use: ST-Link Debugger")
- Check "SWD" is selected in Protocol selection

Keil may reset protocol selection to JTAG when you restart program.
